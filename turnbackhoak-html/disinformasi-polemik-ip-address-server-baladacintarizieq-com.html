(DISINFORMASI): Polemik IP address server baladacintarizieq.com

 

Nama domain baladacintarizieq.com sudah sejak dulu diminta diblokir oleh pemerintah. Jika Anda awam, ketahuilah bahwa mayoritas penyedia layanan internet di Indonesia melakukan intersepsi terhadap permintaan resolve domain name ke mesin DNS mereka sendiri yang sudah dikurasi sehingga banyak menghadang website yang memang sudah dipesan untuk tidak bisa dibuka, dan digantikan dengan tampilan web safe internet.

Sebagai contoh, saya adalah pengguna Indosat GiG, maka hasil resolve akan mengarahkan saya ke webserver Indosat yang menampilkan safe internet (silakan lihat pada gambar tangkapan layar).

Akun twitter @AnonymousNdon yang melakukan nslookup dari komputernya sendiri otomatis juga akan mendapatkan safe internet dari penyedia layanan internet yang dipergunakannya.

Jadi tanpa dia sadari, sebenarnya @AnonymousNdon sedang mengumumkan ke dunia bahwa dirinya adalah pengguna Fastnet. Langkah yang beliau ambil jadi tampak tidak terlalu anonim.

Jika memang ingin pamer memperlihatkan IP address server baladacintarizieq.com, sebaiknya gunakan internet di luar Indonesia agar tidak dicegat saat melakukan resolve. Namun sebenarnya sudah terlambat: nama domain tersebut diurus oleh CloudFlare, dan sekarang sudah tidak bisa di-resolve (perintah “host baladacintarizieq.com” tidak menghasilkan apa-apa).

Jadi klaim Irin Adrina di atas itu tidak benar, dan memang bisa dimaklumi, karena tampaknya beliau awam soal internet, kalau tidak mau dibilang gaptek.
