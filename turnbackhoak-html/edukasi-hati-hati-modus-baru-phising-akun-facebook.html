(EDUKASI): Hati-hati Modus Baru Phising Akun Facebook

 

Modus baru ini dicurigai menggunakan Fans Page sebagai alat untuk menipu target dengan menamakan page #seolah dari ofisial facebook seperti gambar di bawah. Dalam hal ini namanya #Notification_Info.

 

Kalau page tersebut diteliti, postingan-pstingannya hanya berisi tentang peringatan seolah-olah akun target akan dibanned/diblokir oleh facebook.

Cara kerjanya sebagai berikut:

1. FP tersebut akan langsung memposting ke wall target. Begitu target melihat ada posting baru di wallnya, dia berharap target akan mengira itu dilakukan oleh tim resmi facebook.

2. Posting yang berisi peringatan seolah-olah dari facebook tersebut akan meminta korban untuk mengklik sebuah link (yang bukan berdomain facebook dot com) untuk melakukan konfirmasi.

3. Target yang panik atau khawatir sehingga tidak mampu berpikir panjang, tentu bisa saja langsung mengikuti arahan dan memasukkan username + password target di website yang diminta. Sekali target melakukan ini, maka si pencuri telah mendapatkan username dan password target.

Jika anda pernah terjebak dengan hal ini, segeralah ganti password dan email anda untuk mengamankan akun.
