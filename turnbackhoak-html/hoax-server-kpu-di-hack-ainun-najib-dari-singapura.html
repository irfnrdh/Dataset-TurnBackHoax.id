(HOAX): Server KPU di Hack Ainun Najib dari Singapura

Masih terkait HOAX berita KPU dihack yang sedang panas-panasnya dalam isu politik belakangan ini. lagai-lagi kembali beredar berita di media sosial yang tidak jelas asal usulnya seperti pada foto dibawah ini.

berikut adalah bunyi narasinya untuk memperjelas pesan berantai tersebut.

“Ternyata Kepala Administrator server KPU si ainin najib… Team pemenang jokowi di pilpres masa lalu yg tinggal di singapore…
Dan skrg pun ternyata server kpu dikendalikan dr singapore…innalillahi….lah sekarang lembaga negara serevernya dikendalikan dari singapore.. Laa hawla wa laa quwwata illa billaah..”

Lalu ada foto Mas Ainun Najib dengan caption: “Orang yang diduga acak-acak data server KPU. Jika betul agamanya Islam layak…” (foto terpotong).

 

(FAKTANYA):

Ainun Najib BUKAN tim pemenang Jokowi. Dia menggagas Kawal Pemilu bersama empat temannya utk memastikan bahwa perhitungan suara selama Pilpres akurat dengan cara crowdsourcing. Dia juga BUKAN pro Ahok, malah sebaliknya. Dia BUKAN orang yang mengurus server KPU, apalagi terlibat dalam manajemen data elektronik KPU. Dan Ainun Najib adalah Islam sejati.
